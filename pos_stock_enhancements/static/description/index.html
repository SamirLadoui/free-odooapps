POS Stock Enhancements

Real-time Stock & Availability Check in POS by Configurable Location

This module significantly enhances the Point of Sale experience by providing salespersons with real-time stock information based on configurable inventory locations. Prevent overselling, improve inventory accuracy, and boost customer service with immediate stock visibility.

Key Features

Configurable Stock Locations

Easily define which stock locations (warehouses) a specific Point of Sale configuration should check for product availability. Tailor stock checks to the physical location of each POS counter.

Enforce Stock Check Option

Activate or deactivate the real-time stock checking features on a per-POS configuration basis, giving you granular control over inventory management in different sales environments. Note: If enabled, 'Available Stock Locations' MUST be configured.

Real-time Stock Check & Out-of-Stock Warning

View available stock quantities directly on the POS product screen. If a product is out of stock in the configured locations, trying to add it will trigger a clear popup warning, preventing accidental sales of unavailable items.

Automatic Quantity Adjustment

When a salesperson attempts to add a quantity exceeding the available stock for a product, the system automatically corrects the quantity to the maximum available in the configured locations and displays a notification indicating the adjustment.
